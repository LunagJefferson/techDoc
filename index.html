Introduction

HTML stands for HyperText Markup Language. It is used to structure content on the web.

HTML is the foundation of web development, defining the structure of websites.

HTML consists of a series of elements that tell the browser how to display the content.

The first web page, created by Tim Berners-Lee, was a simple HTML document.

Elements

An HTML element usually consists of an opening tag, content, and a closing tag.

HTML elements are the building blocks of a web page. Examples of HTML elements include <div>, <p>, <a>, and <h1>.

<h1>This is a heading</h1>

The above code defines a heading element. Elements can also be self-closing, such as:

<br />

HTML elements can be nested inside one another to create a structure for content.

Attributes

HTML attributes provide additional information about elements.

They are always specified in the opening tag and usually come in name-value pairs, such as:

<a href="https://example.com">Visit Example</a>

The href attribute specifies the destination of the link.

Attributes are case-insensitive, though it is a common practice to write them in lowercase.

Headings

HTML provides six levels of headings, from <h1> to <h6>.

  • <h1> is the largest heading.
  • <h2> is used for subheadings.
  • <h6> is the smallest heading.
  • Headings help organize content and improve accessibility.
  • Search engines also use headings to index the content.
Paragraphs

Paragraphs in HTML are defined with the <p> element.

They are used to separate blocks of text for better readability.

The <p> tag automatically adds space before and after the content, helping to separate the text into logical blocks.

Here's an example of a simple paragraph:

<p>This is a paragraph.</p>
